# Mood-Junkbox

Uppgiften är att skapa en Mood Jukebox. Klicka på knapp för att ändra humör. 


Psuedokod:

- Skapa html strukturen först
- Skapa 3 knappar där man kan välja humör.  <button>Glad</button>

Vid knapptryck:
- ska humöret ändras  
- Namn och artist ska dyka upp 
- Bakgrundsfärg ska ändras

Humör:
- Happy, färg: gul, pharell william - happy
- Sad, färg: blå, Celine dion - All by myself 
- Angry, färg: röd, Rage against the machines - killing in the name
